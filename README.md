# Urdu OCR Project

Urdu Optical Character Recognition (OCR) dataset collection, preprocessing, and model training workflow.

## Project Structure
- `processed_images/`: Contains preprocessed Urdu line text images.
- `labels.csv`: Ground truth text annotations for the dataset.
- `Week2_Saliha.ipynb`: Initial dataset building and preprocessing experiments.
- `Week3_OCR_DataLoader.ipynb`: Expanded dataset (395+ samples) and PyTorch DataLoader integration.
